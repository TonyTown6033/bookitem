# 功能特性说明

## 🎯 核心功能

### 1. 用户管理

#### 功能列表
✅ **用户注册**
- 支持用户名、邮箱、手机号注册
- 密码强度验证（最少6位）
- 邮箱格式验证
- 用户名和邮箱唯一性检查

✅ **用户列表**
- 分页显示用户信息
- 显示用户状态（活跃/禁用）
- 显示注册时间
- 实时数据更新

✅ **用户管理**
- 删除用户功能
- 删除前二次确认
- 级联删除用户的预约记录

#### 数据字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | 字符串 | ✓ | 用户名（唯一） |
| email | 邮箱 | ✓ | 邮箱地址（唯一） |
| phone | 字符串 | ✗ | 手机号码 |
| password | 字符串 | ✓ | 密码（加密存储） |

#### 安全特性
- 🔐 密码 Bcrypt 加密存储
- 🔐 明文密码不返回给前端
- 🔐 用户名和邮箱唯一性约束

---

### 2. 会议室管理

#### 功能列表
✅ **会议室创建**
- 设置会议室名称、位置
- 设置容纳人数
- 添加会议室描述
- 会议室名称唯一性检查

✅ **会议室编辑**
- 修改会议室信息
- 实时保存更新
- 表单数据验证

✅ **会议室列表**
- 展示所有会议室
- 显示会议室状态
- 显示容纳人数和位置
- 支持描述文本省略显示

✅ **会议室删除**
- 删除会议室功能
- 删除前二次确认
- 级联处理相关预约

#### 数据字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | 字符串 | ✓ | 会议室名称（唯一） |
| location | 字符串 | ✓ | 会议室位置 |
| capacity | 整数 | ✓ | 容纳人数（1-500） |
| description | 文本 | ✗ | 会议室描述 |
| is_available | 布尔 | ✓ | 是否可用（默认true） |

#### 业务规则
- 📏 容纳人数必须大于0
- 📏 会议室名称不能重复
- 📏 位置信息必填
- 📏 不可用的会议室无法被预约

---

### 3. 预约管理

#### 功能列表
✅ **预约创建**
- 选择用户和会议室
- 日期时间选择器
- 预约目的填写
- 实时冲突检测

✅ **预约查询**
- 查看所有预约
- 按用户查询预约
- 按会议室查询预约
- 显示预约详细信息

✅ **预约取消**
- 取消已确认的预约
- 取消前二次确认
- 取消后状态更新为"已取消"

✅ **预约删除**
- 永久删除预约记录
- 删除前二次确认
- 从数据库中移除

#### 数据字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| user_id | 整数 | ✓ | 预约用户ID |
| room_id | 整数 | ✓ | 会议室ID |
| start_time | 日期时间 | ✓ | 开始时间 |
| end_time | 日期时间 | ✓ | 结束时间 |
| purpose | 文本 | ✗ | 预约目的 |
| status | 枚举 | ✓ | 状态（pending/confirmed/cancelled） |

#### 业务规则
- ⏰ 开始时间必须早于结束时间
- ⏰ 不能预约过去的时间
- ⏰ 自动检测时间冲突
- ⏰ 会议室不可用时无法预约
- ⏰ 已取消的预约不参与冲突检测

#### 冲突检测逻辑
系统会检测以下情况：
1. 新预约开始时间在已有预约时间段内
2. 新预约结束时间在已有预约时间段内
3. 新预约完全包含已有预约时间段
4. 新预约完全被已有预约时间段包含
